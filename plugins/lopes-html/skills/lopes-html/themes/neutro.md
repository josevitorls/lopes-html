# Tema — Neutro (fallback)

Tema genérico para entregas internas ou sem marca definida.
É a fundação pura, sem overrides de accent: monocromático zinc.

## Overrides de tokens

Nenhum. Valem os padrões de `foundation.md`:

```css
--accent-color: var(--chart-1);   /* #18181b claro · #fafafa escuro */
--accent-soft: rgba(24,24,27,.07); /* claro · escuro: rgba(250,250,250,.10) */
```

## Marca

- Sem nome de marca exibido; título da entrega no lugar do lockup.
- Tom de voz: funcional, curto, operacional (padrão da fundação).
